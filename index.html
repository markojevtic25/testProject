Sentinel is an AI-based product (SaaS) for anomaly detection and prediction that helps monitoring, predictive maintenance, troubleshooting and root-cause analysis for IT hosting providers

Collects system metrics from an existing monitoring system in a non-intrusive way using a plugin-based approach

Applies AI algorithms to system metrics in order to detect and predict anomalies

Generates real-time alerts of detected and/or predicted anomalous states

Offers reports that helps trouble-shooting (root-cause analysis)
